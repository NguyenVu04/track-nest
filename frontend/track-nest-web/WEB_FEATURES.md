Reporter (Phóng viên) Use Cases
REPORT-UC-01 (Publish Missing Person Report): A reporter publishes a missing person report so that users can view it on the system.  

REPORT-UC-02 (Delete Missing Person Report): A reporter deletes a missing person report from the system.  

REPORT-UC-05 (Publish Guideline Document): A reporter uploads and publishes a new crime prevention guideline document to the system.  

REPORT-UC-06 (Delete Guideline Document): A reporter deletes an outdated crime prevention guideline document from the system.  

REPORT-UC-09 (Publish Crime Report): A reporter adds and publishes a new crime report.  

REPORT-UC-10 (Delete Crime Report): A reporter deletes an outdated crime report from the system.  

2. Emergency Service (Dịch vụ cứu hộ) Use Cases
REPORT-UC-11 (Generate Crime Analysis Report): Emergency services generate a synthesized report on the crime situation in specific areas, which is then displayed on the dashboard and available for download.  

EMERGENCY-UC-02 (Reject an Emergency Request): A rescue worker rejects a rescue request when they are unable to accept it due to overload, being out of the support range, or having incomplete information.  

EMERGENCY-UC-03 (Accept an Emergency Request): Emergency services accept an emergency request to begin the process of assisting the person in distress.  

EMERGENCY-UC-04 (Complete an Emergency Request): Emergency services mark a rescue request as processed and completed after successfully finishing the assistance task.  

EMERGENCY-UC-05 (Remove Safe Zone): Emergency services remove a safe zone (such as a police station) from the system.  

EMERGENCY-UC-06 (Add Safe Zone): Emergency services add a new safe zone (such as a police station) into the system.
