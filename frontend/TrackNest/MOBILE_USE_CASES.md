1. Tracking Features
TRACK-UC-00 (Track Targets): The user uses an interactive map to track the real-time location of themselves and their relatives within the same circle.  

TRACK-UC-01 (Receive Target Mobility Anomaly Alerts): The user receives notifications when the system detects that the tracked person deviates far from their normal habits or safe zones.  

TRACK-UC-02 (Enable Tracking): The user enables the location sharing feature to allow people in their corresponding circle to track them in real-time.  

TRACK-UC-02 (Disable Tracking): The user actively turns off the location sharing feature, stopping the collection and transmission of location data to the circle.  

TRACK-UC-03 (Create Circle): The user creates a new family circle and takes on the role of "Guardian".  

TRACK-UC-04 (Join Circle): The user enters a code to join an existing circle and by default becomes a "Tracked Person".  

TRACK-UC-05 (Leave Circle): The user actively leaves a family circle they have joined.  

TRACK-UC-06 (Remove From Circle): The user acting as an "Admin" (Guardian) has the right to remove another member from the circle.  

TRACK-UC-07 (View Mobility History): The user looks up the location and movement history of themselves or linked relatives directly on the map.  

2. Report Features
REPORT-UC-00 (Submit a Missing Person Report): The user creates and submits a report about a missing person to the system.  

REPORT-UC-03 (Delete Submitted Missing Person Report): The user deletes a missing person report that they previously submitted themselves.  

REPORT-UC-04 (View Guideline Document): The user accesses and reads crime prevention guideline documents provided on the mobile application.  

REPORT-UC-07 (View Missing Report): The user looks up and views the details of missing person reports that have been published.  

REPORT-UC-08 (View Crime Report): The user views the list and details of criminal activities occurring in nearby areas.  

REPORT-UC-12 (Receive High-Risk Crime Zone Alerts): The user receives automatic alerts when entering geographic areas with a high crime density.  

3. Emergency Features
EMERGENCY-UC-00 (Activate Emergency Alert By Button): The user presses the "SOS" button to send a danger alert along with their last known location to relatives and rescue services.  

EMERGENCY-UC-01 (Activate Emergency Alert By Voice): The user speaks pre-set keywords (e.g., "Help") to trigger a hands-free SOS alert.  

EMERGENCY-UC-07 (Find Safe Zone): The user searches for nearby safe locations (like police stations) directly on the application's map.
