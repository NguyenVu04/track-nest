# Location Tracking Migration

This folder contains migration scripts and resources for integrating location tracking data with MinIO storage. Use these files to manage, transfer, and update location tracking information as part of the migration process.

## Usage

1. Review the migration scripts and configuration files.
2. Update any necessary parameters (e.g., MinIO endpoint, access keys).
3. Run the migration scripts to move data between local storage and MinIO.

## Notes

- Ensure MinIO is running and accessible before starting migration.
- Backup existing data before performing any migration.
- Refer to individual script documentation for detailed instructions.
